# VisitTheUSA Design Reference

## Scraped 2026-04-16

### Homepage Layout Pattern
- Full-screen hero with transparent header overlay
- "Thinking About a Trip?" intro section (cream bg)
- **6 full-width hero carousel cards** (1440x720px each) — horizontal slider with 01/06 numbering
- Each card: full-bleed image + content area (768px wide, 48px padding)
- UGC / social section
- Interactive map section
- Common slider (destination cards)
- FAQ accordion (dark bg)
- Footer: background image (550px) + newsletter signup + bottom bar

### Card Structure
- Image: full-bleed background cover
- Content area: 768x720px, 48px padding, left-aligned text
- Category label: 24px, weight 350, white, uppercase, 4px letter-spacing
- H2: 96px, weight 900, white, uppercase
- Body: 20px, weight 350, white, 30px line-height
- CTA button: pill shape, 16px, weight 700
